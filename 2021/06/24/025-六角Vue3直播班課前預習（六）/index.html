紀錄六角網頁Vue3直播班課前預習筆記（Vue Cli）

Vue Cli

為什麼要用Vue Cli

  1. 前端開發日趨複雜，整合性的工具可以減少彼此開發上的差異
  2. 編譯環境越來越複雜，使用整合性工具建立環境
  3. 前後端分離形成主流
  4. 套件引用好管理

Vue Cli是什麼

  1. 基於Webpack所建置的開發工具
  2. 便於使用各種第三方套件（BS4、Vue Router等等）
  3. 可運行Sass、Bebel等編譯工具
  4. 獨特Vue檔案，一口氣包含html、js、css
  5. 便於開發SPA的網頁工具

Vue Cli開始流程

  1. 安裝NodeJs
  2. 建立Vue Cli全域環境
  3. 建立Vue專案
  4. 開始開發

Vue Cli相關資源

  1. NodeJs
  2. Vue Cli
  3. Vue Router
  4. Axios
  5. Vue Axios

課程中用到的指令

  • 建立專案：vue create {{ 專案名稱 }}
