V8、Node.js介紹

V8

  • JavaScript引擎
  • google團對釋放
  • chrome內涵V8
  • Node.js依循V8規則開發

Node.js架構介紹

  • Node.js Github 連結

命令提示字元操作教學

  1. 尋找電腦裡面終端機（命令提示字元）
    MAC：termianl.app(cmd+空白鍵)
    WIN：cmd(開始->尋找)
  2. 熟悉指令
    cd 移動路徑
    cd .. 回到上一層
    ls 瀏覽所在目錄（MAC）
    dir 瀏覽所在目錄（WIN）

安裝Node.js

  1. 安裝nvm進行版本切換（碰到重開就要重裝的解決方式）
  2. nvm指令
    nvm install [node 版本號] - 安裝特定版本的 nvm
    nvm list - 列出已安裝清單
    nvm use [node 版本號] - 切換到特定 Node 版本(只限有安裝的版本)
  3. node指令
    node –version

使用node開始編譯

  1. 輸入node
  2. 結束時輸入兩次ctrl+c

使用node執行js檔案

  1. cd到檔案資料夾
  2. node [檔案名稱]

Global 全域物件

  1. 瀏覽器有全域物件window
  2. node有全域物件global
  3. 全域變數會負寫到window，node裡則不會
