VsCode的安裝套件

  • Chinese (Traditional) Language Pack for Visual Studio Code
    筆者假美國人，英文不好先裝中文介面就對了
  • Auto Rename Tag
    更動標籤其中一側時，將另一側自動更改
  • Live Sass Compiler
    sass編譯器，將sass/scss編譯成css
  • Live Server
    即時預覽網頁
  • Material Icon Theme
    將左側檔案總管依檔案類型放置不同的icon，方便辨識
  • Material Theme
    技不如人也要有比別人漂亮的介面裝逼

position定位

消除div包img時多些微px的策略

  1. 把 img 改成 inline-block 以外的排版方式
    這個方式相對容易理解，可以改成 display: block，或是float: left
  2. 在維持 inline-block 的情況下，抹除那些空白帶來的空間
    這部份理解就稍微複雜，像是在父元素 .headerLogo 上，添加一些 CSS 設定。
    例如 font-size: 0px; ，或是 line-height: 0，這些都能移除 inline-block 額外添加的空白效果
